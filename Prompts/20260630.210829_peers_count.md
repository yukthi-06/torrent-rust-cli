# Implementation
- Added `peers_connected: usize` to `TorrentState` in `crates/daemon/src/server.rs`.
- In `crates/daemon/src/engine.rs` `TorrentDownloader::handle_peer`, added logic to increment `peers_connected` upon a successful handshake.
- Added a `Drop` guard to decrement `peers_connected` automatically when the peer connection is closed or drops.
- Updated `Request::List` and `Request::Status` in `crates/daemon/src/server.rs` to return the live peer count.

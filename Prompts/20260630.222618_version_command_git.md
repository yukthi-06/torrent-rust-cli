# User Prompt
> version command should also show the git SHA full code and date of commit

# Implementation
- Created `crates/cli/build.rs` and `crates/daemon/build.rs` to extract Git hash and date using `git rev-parse HEAD` and `git log -1 --format=%ci`.
- Exposed these as `GIT_HASH` and `GIT_DATE` environment variables to rustc.
- Updated `crates/cli/src/main.rs` and `crates/daemon/src/server.rs` to include the git hash and date in the version strings.
